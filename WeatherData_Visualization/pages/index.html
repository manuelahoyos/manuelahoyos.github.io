Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator. The data used was pulled from the OpenWeatherMap API to assemble a dataset of 546 cities.

Matplotlib was used plot the various aspects of weather in the dataset vs. latitude. These factors include temperature, cloudiness, wind-speed and humidity. This website provides the source data and the graphs created in the analysis, as well as the explanations and descriptions of any trends.
